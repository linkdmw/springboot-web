# Springboot-Web

自用学习，一个简单的使用springboot开发的简单WEB项目demo，实现了基本的登录退出以及一个模块的基本增删改查；

### 技术栈

-   SpringBoot 2.3.2
-   Thymeleaf+BootStrap
-   MySQL 8.0
-   Shiro
-   Swagger
